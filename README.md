一,此包为整个项目的压缩包,其中包括.java源文件，字节码文件，相关图片资源，以及数据库表

二，数据库采用mysql，用户名root，密码为空，其中包括两个表，一个是用户信息，一个好友信息；

三，服务器IP：localhost  ，  端口：5000

四，运行时，请先导入数据库文件，然后启动服务器（serverQQ），随后运行登入（Login)
